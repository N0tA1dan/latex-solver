# LaTeX Solver
LaTeX Solver is a mathematical computational engine built for solving integrals, limits, and algebraic expressions.

## How it works

We take an expression in LaTeX format, build an AST (Abstract Syntax Tree), then walk the tree and apply certain rules/theorems for the given expression.


## Requirements

Cmake 3.5 or above


## Installation

Clone this repository then run the following:

`mkdir  build && cd build`
`cmake ..`

then to run do 
`./solver`
